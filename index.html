HTML

  • Each HTML element performs a unique role, But the two most important elements to understand how the structure works are head and body elements.
  • The head element contains information about webpages, kind of like a title.
  • The body element represents the visible content shown to the user, like text following the title in an article.
  • HTML is used to create the structure of a webpage.
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets (<>).

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The preferred in most cases is External CSS style sheet because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • When using an external style sheet, we link the file using a element placed in the HTML's
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • Inline CSS styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
